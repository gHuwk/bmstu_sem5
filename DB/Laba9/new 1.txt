Написать на языке Visual C# консольное приложение с консольным меню, состоящее из 10 функций, демонстрирующих основные приемы работы с ADO.NET: 
	5 функций работы с отсоединенными объектами и 
	5 функций работы с присоединенными объектами.
Исходные данные для параметризованных запросов и пераметризованных модулей вводить с клавиатуры. 
Контроль ввода проводить в блоках try-catch с использованием класса FormatException.
 Для преобразования данных использовать статические методы класса Convert и статический метод Parse для  типов данных.
 Для задания параметризованных запросов и пераметризованных модулей можно использовать  метод Format класса String.

Примечание
При написании приложений в первую очередь руководствоваться примерами из папки [Примеры для подражания]. В этой папке все примеры разделены на две группы:
 1.	Работа с подсоединенными объектами:
 a)	Primer01 — проверка строки подключения, 
 b)	Primer02 — выполнение скалярного запроса,
 c)	Primer03 — выполнение запроса, возвращающего набор строк,
 d)	Primer04 — выполнение параметризованных запросов на обновление,
 e)	Primer05 — выполнение хранимой процедуры без параметров,
 f)	Primer06 — выполнение хранимой процедуры с параметрами.
 2.	Работа с отсоединенными объектами:
 a)	Primer07 —  наполнение таблицы набора данных результатами запроса,
 b)	Primer08  — работа с таблицами набора данных (фильтрация и сортировка),
 c)	Primer09 —  обновление данных в базе данных через адаптер данных,
 d)	Primer10  — сохранение данных набора данных в виде  XML-файла.
При желании можно руководствоваться альтернативными примерами из папок [ПримерыN].
